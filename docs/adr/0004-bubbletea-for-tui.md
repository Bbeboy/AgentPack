# 0004 - Use Bubble Tea for the future TUI

## Context

The roadmap includes a TUI after the MCP server. The prompt adapter decision already aligns the project with the Charm ecosystem.

## Decision

Adopt Bubble Tea for the future TUI implementation.

## Consequences

- Prompt and TUI work can share ecosystem conventions.
- The TUI can be added as another input adapter without changing application use cases.
- Terminal interaction choices made in earlier phases should stay compatible with Bubble Tea expectations.
